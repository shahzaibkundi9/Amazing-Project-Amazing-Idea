# WhatsApp AI Automation (Monorepo)

This project includes:

- Backend (Node + Express + Prisma + PostgreSQL)
- AI Service (Multi-AI Provider Rotation Engine)
- WhatsApp Automation Service (Baileys MD)
- Admin Dashboard (Next.js 14)
- Cloudinary Uploads
- Real-time Chat Sync (Socket.io)
- Payment Screenshot Pipeline

## Monorepo Folders

- /backend
- /frontend
- /ai
- /whatsapp

## Development Commands

pnpm dev:all  
pnpm dev:backend  
pnpm dev:ai  
pnpm dev:whatsapp  
pnpm dev:frontend  

## Build

pnpm build 

## Environment

Configure each service's `.env` file as required.
